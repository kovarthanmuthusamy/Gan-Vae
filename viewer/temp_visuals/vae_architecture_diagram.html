📊 Encoder Architecture:

  • 🎯 Design: Separate encoding of normalized heatmap and max impedance
  • 📡 MaxImpedanceBranch: MLP 1→256→512→8192, reshaped to 8x8x128
  • 🔥 HeatmapBranch: CNN 64x64x2 → 8x8x128
  • 🗺️ OccupancyBranch: Conv at 7x8, upsample to 8x8x128
  • 📊 ImpedanceBranch: MLP 231→512→8192, reshaped to 8x8x128
  • 🔗 Late Fusion: Concat all branches (512 ch) → Conv to 128 ch → Self-attention → Latent 128D

📊 Decoder Architecture:

  • 📡 MaxImpedanceDecoder: MLP 128→128→64→1, outputs scalar max_impedance (independent head)
  • Master Grid: FC 128→8192, upsample 4x4→16x16x128 with self-attention
  • 🎯 HeatmapDecoder: Independent decoder from master grid, outputs 64x64x2 with sigmoid (no max_impedance conditioning)
  • 🗺️ OccupancyBranch: Adaptive pooling to 7x8, outputs logits
  • 📊 ImpedanceBranch: MLP 128→256→512→231
  • 🔗 Physical Reconstruction: heatmap_physical = heatmap_norm × max_impedance (computed in loss, not during decoding)

📊 Pipeline Details:

  • 🗂️ Dataset Preparation: Per-sample normalization, heatmap_norm in [0,1], max value stored separately
  • 🏋️ Training: DataLoader loads heatmap_norm + max_value. Decoder has 4 independent heads. Loss computed on physical scale
  • 🔮 Inference: Saves heatmap_norm and heatmap_physical
  • 🎯 Design: Separate learning, numerical stability, physical consistency, independent decoder heads

📊 Loss Function Details:

  • 🎯 Physical Heatmap Loss: Reconstruct physical heatmap and compute MAE. Couples normalized heatmap with max impedance via gradients. Uncertainty weighted with learnable σ_h
  • 📡 Max Impedance Loss: MAE for scalar prediction accuracy. Uncertainty weighted with learnable σ_m
  • 🗺️ Occupancy Loss (Triple): Combines Dice + Focal + Weighted BCE for class imbalance (6-8% occupied). Static weight (5.0) for first 50 epochs, then uncertainty-based
  • 📊 Impedance Loss: MAE on 231D vector. Uncertainty weighted with learnable σ_i
  • 🧬 KL Divergence: Regularizes latent to N(0,1). β-annealing from 0.0→1.0 over epochs 0-50 to prevent collapse
  • ⚖️ Uncertainty Weighting: Automatic task balancing via learned σ parameters. Hard tasks get lower weight, easy tasks get higher weight
  • Total: Weighted sum of all losses with gradient clipping for stability
